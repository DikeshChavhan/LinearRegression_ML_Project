# Insurance Charges Predictor Streamlit App

This package contains a Streamlit app that uses the uploaded model file `insurance_model.pkl`.

Files:
- app.py
- utils.py
- requirements.txt
- data/data_sample.csv
- insurance_model.pkl

To run:
pip install -r requirements.txt
streamlit run app.py

Note: Preprocessing is a best-effort and may need to match training preprocessing exactly for accurate results.
